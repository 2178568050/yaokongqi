# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 最新 main 分支 | ✅ 接受报告 |
| 0.1.x 发布版 | ✅ 接受报告 |
| 更早版本 | ❌ 不保证修复 |

## 报告方式

**请勿**在公开 Issue 中披露可被利用的漏洞细节。

请发送邮件至：**2178568050@qq.com**

邮件标题建议：`[Security] Yaokongqi - 简要描述`

请包含：

- 漏洞类型与影响（如：未授权输入注入、PIN 暴力破解等）；
- 复现步骤与环境；
- 如有：PoC、建议修复思路。

## 响应时间

作者将在 **7 个工作日内**确认收到；严重问题优先处理。

## 安全使用建议

- 仅在可信局域网使用；
- 定期更新至最新版本；
- 不使用时关闭 PC 端服务；
- 勿将 PIN 分享给他人。
