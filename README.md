## 产品介绍

**遥控器**是一款面向 Windows 用户的局域网远程输入工具：在 Android 手机上安装客户端，在 PC 上运行轻量托盘服务，通过同一 WiFi 即可把手机当作**无线触摸板 + 场景化按键面板**使用。

无需蓝牙配对器、无需公网账号、无需在 PC 安装大型远程桌面软件。配对完成后，你可以：

- 躺在沙发上用触摸板控制电脑光标；
- 一键切换「刷视频 / 办公 / 演示 / 游戏」等按键布局；
- 在竖屏输入中文，自动写入 PC 剪贴板并粘贴；
- 用多指手势切换窗口、打开任务视图。

PC 端采用 Rust 编写，托盘常驻、空闲低占用；Android 端采用 Kotlin + Jetpack Compose，支持深色模式、横竖屏分栏与多套内置场景方案。

**适用场景**：客厅 PC 遥控、演示汇报翻页、短视频刷流、轻办公快捷键、躺平浏览网页等。

**不适用场景**：跨网远程办公（需同一局域网）、高帧率游戏精确操作、对延迟极度敏感的专业设计工作。

## 功能特性

### 连接与安全

| 功能 | 说明 |
|------|------|
| 局域网 WebSocket | 端口 `10825`，低延迟，数据不出路由器 |
| PIN 配对 | 6 位配对码，5 分钟有效，过期自动刷新 |
| 设备记忆 | 配对成功后自动重连，无需重复输入 PIN |
| 多设备历史 | 可保存多台 PC，在连接页快速切换 |

### 触摸板

| 手势 | 作用 |
|------|------|
| 单指滑动 | 移动鼠标（可调低/中/高灵敏度） |
| 单指点击 | 左键单击（双击间隔内再点 = 双击） |
| 双指滑动 | 滚轮滚动 |
| 双指点击 | 右键 |
| 三指轻点 | 任务视图（部分机型三指滑动被系统占用） |
| 四指左右滑 | 切换窗口（Alt+Tab / Alt+Shift+Tab） |
| 四指上/下滑 | 任务视图（Win+Tab） |
| 双指外扩 | 进入极简滚轮模式（可关闭） |
| 竖滑滚轮模式 | 可选：单指竖滑代替滚轮 |

### 多场景按键布局

内置 **7 套**场景方案，在按键与触摸板之间的空白区域滑动即可切换（竖屏左右滑 / 横屏上下滑）：

| 方案 | 说明 |
|------|------|
| 刷视频 | 音量±、←→ 切条、↑↓ 翻页（4×2 布局） |
| 打字 | Esc / 翻页 / Shift / Win / Delete / 唤起键盘 / Enter |
| 办公编辑 | 复制、粘贴、撤销、保存、查找、全选 + 键盘 |
| 演示汇报 | ← → 翻页、F5 放映、Esc 退出 |
| 网页浏览 | 前进后退、刷新、标签页、PageUp/Down、空格 |
| 窗口切换 | Alt+Tab、Win+D/E/L、Alt+F4、键盘 |
| 游戏方向 | 方向键 + 空格 |

- 内置方案**可重命名**，按键布局只读；可**另存为自定义方案**后自由编辑
- 支持 2×2 / 3×3 / 4×2 / 4×4 网格、单键、双键、全键盘等布局模式
- 横屏**分栏布局**：左侧按键、中间留白、右侧触摸板

### 文本输入

- 点击顶栏键盘图标或布局中的「唤起键盘」
- 竖屏：全屏输入页；横屏：底部浮层
- 支持**中文**：写入 PC 剪贴板并模拟 Ctrl+V 粘贴

### 其他

- 深色模式、按键震动反馈
- 连接质量显示（延迟、丢包率）
- 音量加键连按保护（防误触确认）
- 极简滚轮模式：全屏只保留滚轮，长按退出
- PC 托盘：显示 IP / PIN、刷新配对码、开机自启、关于与法律信息

## 使用方法

### 准备条件

- **PC**：Windows 10 / 11（64 位）
- **手机**：Android 8.0+（API 26）
- 两者连接**同一 WiFi 或同一局域网**
- PC 防火墙首次运行时允许「**专用网络**」访问

### 第一步：启动 PC 端

**方式 A — 使用发布包（推荐）**

1. 打开 [`dist/遥控器.exe`](dist/遥控器.exe)（或从 [Releases](https://github.com/2178568050/yaokongqi/releases) 下载）
2. 看系统托盘（任务栏右下角 ^）出现蓝色图标
3. **鼠标悬停**图标，记录显示的：
   - 局域网 IP（如 `192.168.1.100`）
   - 端口 `10825`
   - 6 位 **PIN**（5 分钟内有效）

**方式 B — 从源码编译**

```powershell
cd pc
cargo run --release
```

> 日志文件：`%APPDATA%\Yaokongqi\yaokongqi.log`

### 第二步：安装并打开手机端

**方式 A — 使用发布包**

安装 [`dist/遥控器.apk`](dist/遥控器.apk)（或 Releases 中的 APK）。

**方式 B — 从源码编译**

```powershell
cd android
.\gradlew assembleDebug
# 安装 app/build/outputs/apk/debug/app-debug.apk
```

### 第三步：配对连接

1. 打开手机 App，在连接页输入 PC 的 **IP 地址** 和 **PIN**
2. 点击配对；成功后显示 PC 名称与连接状态
3. 之后同一 PC 会自动重连，一般无需再输 PIN

### 第四步：日常使用

竖屏界面自上而下分为四块：

| 区域 | 高度占比 | 操作 |
|------|----------|------|
| 顶栏 | — | 方案名、连接延迟、文本输入 / 设置 / 断开 |
| 自定义按键 | 约 42% | 点击发送快捷键（复制、翻页、音量等） |
| 空白区 | 约 12% | **竖屏左右滑** / **横屏上下滑** 切换场景方案 |
| 触摸板 | 约 46% | 单指移动、双指滚轮/右键、多指切窗 |

横屏分栏（需在设置中开启）：左侧按键 · 中间空白区上下滑切换方案 · 右侧触摸板。

1. **移动光标**：在触摸板区域单指滑动
2. **点击 / 右键**：单指轻点 / 双指轻点
3. **滚轮**：双指上下滑（或在设置中开启「竖滑滚轮」）
4. **按快捷键**：点按上方按键（如复制、翻页、音量）
5. **切换方案**：在按键与触摸板之间的**空白区域**滑动切换
6. **输入文字**：点顶栏键盘图标，输入后发送到 PC
7. **调整设置**：顶栏齿轮 → 灵敏度、布局、深色模式等

### 托盘菜单（PC）

右键托盘图标：

| 菜单项 | 说明 |
|--------|------|
| 刷新配对码 | 生成新 PIN（新设备配对时使用） |
| 开机自启 | 勾选后登录 Windows 时自动启动 |
| 关于与法律信息 | 许可、免责声明、作者邮箱 |
| 退出 | 关闭服务 |

### 常见问题

| 问题 | 处理 |
|------|------|
| 手机连不上 | 确认同一 WiFi；IP 填 `192.168.x.x` 不要填 `127.0.0.1`；检查防火墙 |
| PIN 无效 | PIN 5 分钟过期，看托盘悬停信息或点「刷新配对码」 |
| 触摸板不跟手 | 设置 → 触摸灵敏度调「高」 |
| 三指手势无效 | 多为系统截图占用，可试四指手势或按键方案 |
| 杀毒拦截 | 绿色 exe 可能被误报，添加信任即可 |

更详细的 PC 说明见 [`dist/PC端-使用说明.txt`](dist/PC端-使用说明.txt)。

## 开发者快速开始

### 项目结构

| 目录 | 说明 |
|------|------|
| [`android/`](android/) | Android 客户端（Kotlin + Compose） |
| [`pc/`](pc/) | Windows 服务（Rust + 系统托盘） |
| [`docs/`](docs/) | 协议、开发、法律文档 |
| [`dist/`](dist/) | 本地发布包（构建产物，见 `.gitignore`） |

### 构建发布包

```powershell
# Android
cd android
.\gradlew assembleDebug

# PC
cd pc
cargo build --release

# 复制到 dist/
Copy-Item android\app\build\outputs\apk\debug\app-debug.apk dist\遥控器.apk
Copy-Item pc\target\release\yaokongqi.exe dist\遥控器.exe
```

协议与端口详见 [docs/protocol.md](docs/protocol.md)，开发说明见 [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md)。
