# 射击游戏模式使用说明

本文介绍「遥控器」**射击游戏模式**：用手机横屏模拟 **Xbox 360 虚拟手柄**，在 PC 上玩支持手柄的射击游戏（如 Apex Legends）。

> **说明**：本模式通过 WiFi 将触摸输入转为虚拟手柄信号，**不是** reWASD 类 PC 端键鼠重映射；游戏内请使用**手柄灵敏度**设置。

---

## 准备条件

| 项目 | 要求 |
|------|------|
| 手机 | Android 8.0+，建议横屏使用 |
| PC | Windows 10/11，已运行遥控器托盘服务 |
| 网络 | 手机与 PC 同一局域网；PC 建议网线，手机连 **5GHz WiFi** |
| 驱动 | PC 安装 [ViGEmBus](https://github.com/nefarius/ViGEmBus/releases)（虚拟手柄驱动） |
| 游戏 | 拔掉实体手柄，游戏内选手柄操作方案 |

连接成功后，游戏模式顶部居中会显示 **RTT 延迟**（如 `18ms`），竞技建议 **&lt;30ms**、无丢包。

---

## 如何进入

1. 手机与 PC 完成配对连接。
2. 打开 **设置 → 射击游戏（虚拟手柄）**。
3. 开启 **射击游戏模式**，按需调节灵敏度与布局。
4. 点击 **保存设置**，返回主界面即进入横屏游戏界面。
5. 左上角可 **编辑按键布局**；设置内可 **退出游戏模式** 回到普通遥控界面。

---

## 操作分区

横屏下屏幕分为左右两半：

| 区域 | 操作 |
|------|------|
| **左半屏** | 左下 **移动轮盘**：控制角色移动；摇杆**上推**可锁定奔跑（LS） |
| **右半屏** | **滑动瞄准**：手指在右半屏任意位置拖动即可转向/压枪（无需对准固定区域） |
| **屏幕按键** | 开火、开镜、跳跃、技能等，位置可在编辑模式中拖动调整 |

游戏中瞄准区**不可见**（纯黑画面），编辑布局时会显示右半屏参考框。

---

## 默认按键映射（Apex 预设）

屏幕按键通过 ViGEm 映射为 Xbox 360 输入：

| 屏幕按键 | Xbox / 游戏语义 |
|----------|-----------------|
| 射击 | 右扳机 RT |
| 开镜 | 左扳机 LT |
| 跳跃 | A |
| 蹲下 | B |
| 换弹 / 互动 | X（与 Apex 手柄上下文一致） |
| 战术技能 | LB |
| 绝招 | RB |
| 投掷 | 方向键右 |
| 治疗 | 方向键上 |
| 求生 | 方向键左 |
| 切枪 | Y |
| 地图 | Back |
| 背包 | Start |

可在 **编辑按键布局** 中调整位置、大小与单键透明度；**恢复 Apex 默认布局** 可一键还原。

## 灵敏度与手感设置

在 **设置 → 射击游戏模式** 中可调节：

| 设置项 | 说明 | 竞技参考 |
|--------|------|----------|
| 输入刷新率 | 手柄状态发送频率 | **180~250 Hz** |
| 移动灵敏度 | 左轮盘输出倍率 | 1.0~1.2 |
| 腰射水平/垂直灵敏度 | 右半屏未开镜时的瞄准 | 水平 3~6（小屏可更高），垂直 1.2~1.8 |
| 开镜水平/垂直灵敏度 | **按住开镜键时** 单独生效 | 通常低于腰射 |
| 滑动加速度 | 快速滑动额外加速，便于转身 | 1.5~3；0 为匀速 |
| 瞄准平滑过滤 | 越大越稳，越小越跟手 | 0.05~0.12 |
| 移动死区 | 左轮盘中心无响应区 | 0.06~0.10 |
| 瞄准松手衰减 | 松手后准星惯性，越大停得越慢 | 要快停设 0.55~0.62 |
| 控件透明度 | 全局；单键可在布局编辑中覆盖 | 0.35~0.5 |
| 非固定移动轮盘 | 关闭=左下固定轮盘（推荐） | 建议关闭 |

**开镜灵敏度**：按住屏幕「开镜」键时，右半屏滑动自动切换为开镜水平/垂直灵敏度。

**小屏建议**（约 6.9 寸横屏）：优先提高 **腰射水平灵敏度** 与 **滑动加速度**；开镜单独压低，避免瞄头过冲。

## 布局编辑

1. 设置内点 **编辑按键布局**，或游戏界面左上角进入编辑。
2. **拖动** 按键与移动轮盘调整位置。
3. 点选按键后，左下角滑条调节 **大小** 与 **透明度**。
4. 点 **完成** 保存布局。

移动轮盘与右半屏瞄准区为固定逻辑分区，按键可自由摆放。

---

## 游戏内设置建议（Apex）

1. 控制器类型选手柄，**拔掉实体手柄**。
2. 视角灵敏度 / ADS 灵敏度：先用游戏默认，再用 App 微调；避免两边都拉满。
3. 响应曲线可试 **Classic** 或 **Linear**。
4. 在训练场练习：转身 → 开镜跟枪 → 边移动边射击，再回 App 细调。

## 常见问题

| 问题 | 处理 |
|------|------|
| 提示「虚拟手柄不可用」 | PC 安装 ViGEmBus 后重启托盘服务 |
| 游戏无反应 | 确认已连接；射击模式已开启并保存；游戏识别到手柄 |
| 转身慢 / 瞄不准 | 调高腰射水平灵敏度或滑动加速度；检查 RTT |
| 开镜太抖 | 降低开镜水平灵敏度或滑动加速度 |
| 延迟高 | PC 插网线、手机近路由器；顶部 RTT &gt;50ms 时避免排位 |
| 与 reWASD 关系 | 仅 PC 端「虚拟 Xbox 手柄」输出类似；本方案是手机 WiFi 远程手柄 |

```
手机触摸 → WebSocket → PC 托盘 → ViGEm 虚拟 Xbox 360 → 游戏
```
