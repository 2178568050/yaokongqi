## 采用什么许可证？

本项目源代码以 **[GNU AGPL v3.0](../LICENSE)** 发布。

| 特点 | 说明 |
|------|------|
| 可自由使用 | 个人、学习、研究、非盈利场景 |
| 可 Fork | 必须开源衍生代码，并保留版权与许可证 |
| 网络服务 | 若基于本代码提供网络服务，须向用户提供源码（AGPL 特色） |
| 专利 | 贡献者授予用户必要专利许可 |

## 为什么选 AGPL？

- **对开发者友好**：欢迎 Fork、PR、二次开发，社区可共建；
- **对闭源商用有约束**：若大厂直接拿去改一改闭源卖，AGPL 要求公开修改后的源码；
- **对 SaaS 有约束**：基于本代码的在线服务也须满足源码提供义务。
