# 使用规范

## 允许的使用

| 场景 | 说明 |
|------|------|
| 个人使用 | 在自己手机与 PC 间远程控制，免费 |
| 学习研究 | 阅读、学习、调试源码 |
| 教学演示 | 课堂/技术分享中演示（注明出处） |
| 开源二次开发 | Fork 后**保持 AGPL-3.0 开源**，保留版权与修改说明 |
| 非盈利组织 | 内部非商用部署（如学校机房、创客空间） |

## 禁止的使用

| 场景 | 说明 |
|------|------|
| 未经授权商用 | 收费销售、订阅、广告变现、企业 SaaS 等 |
| 闭源衍生 | 基于本代码开发闭源产品且不公开修改后的源码 |
| 去除版权 | 删除 LICENSE、NOTICE、作者信息或开源声明 |
| 恶意用途 | 未授权控制他人设备、传播恶意软件、绕过安全机制 |
| 品牌冒用 | 以「遥控器」名义发布未授权版本造成用户混淆 |

## 二次开发要求（Fork）

若您基于本项目进行二次开发并**公开分发**（含 GitHub 发布、APK 分享等），请：

1. **保留** LICENSE、NOTICE 及本仓库法律相关文档；
2. **标注**您对代码的修改（提交记录、CHANGELOG 或 README 说明）；
3. **开源**您的衍生作品（AGPL-3.0 要求）；
4. **修改**应用名称与图标，避免与原版「遥控器」混淆（见 [TRADEMARK.md](TRADEMARK.md)）；
5. 若您的衍生作品涉及**商用**，须先取得作者书面授权。

## PC 端特别说明

- 首次运行请允许 Windows 防火墙「专用网络」访问；
- 不使用时建议退出托盘程序，减少局域网暴露面；
- PIN 配对码有效期 5 分钟，请勿将 PIN 分享给不可信人员。

## Android 端特别说明

- 需与 PC 处于同一局域网；
- 文本输入功能会将内容写入 PC 剪贴板并模拟粘贴，请勿输入敏感密码（建议使用 PC 本地输入）。
