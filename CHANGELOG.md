# 更新日志

本文件记录「遥控器」(Yaokongqi) 的版本变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [0.2.6] - 2026-06-13

### 新增

- **手柄二进制 WebSocket 帧**：18 字节固定布局，替代 JSON `gamepad` 消息（PC 仍兼容旧版 JSON）。
- **UDP 低延迟数据通道**：`udp://PC:10826`，配对 / pong 下发会话密钥；设置中可开关。
- **500 Hz 发送选项**：设置 → 射击游戏可选 120 / 180 / 250 / **500** Hz。

### 改进

- **瞄准即时上报**：触摸增量后立即 UDP/WS 发送，不再等发送周期 tick。
- **手柄全控件即时上报**：冲刺、轮盘、战术键、瞄准松开等全部立即 flush。
- **遥控器触摸板即时移动**：取消 2ms 批量与位移阈值；使用 historical 高采样；手势结束刷余量。
- **瞄准按住维持偏转**：右摇杆改为瞬时位移映射 rx/ry，按住期间保持直至松手（非脉冲清零）。
- Android 瞄准输出取消死区；PC 右摇杆死区降至 **24**。
- **合并手柄发送循环**：移除 ConnectionManager 冗余轮询，仅保留 MainViewModel 单循环（瞄准手势进行中 tick 不再插入 rx/ry=0）。
- PC ViGEm 线程改为 **100µs 事件唤醒**（原 1ms sleep），收包后更快 push 虚拟手柄。
- PC ViGEm 刷新上限提升至 **500 Hz**；Android 发送间隔最小 **2 ms**。
- **设置页调参优化**：移除已废弃的「瞄准衰减」；默认值与滑条区间适配「按住维持偏转」模型；分组为移动 / 腰射 / 开镜 / 开火 / 高级；新增「恢复灵敏度默认值」。
- **瞄准静止衰减**：手指停住约 45ms 后 rx/ry 逐步归零（设置 → 高级，默认开启）。
- **连接页快捷入口**：已连接时显示「进入射击模式」，一键开启虚拟手柄。
- **PC 手柄路径优化**：ViGEm 相同状态跳过 update、合并高频 Update 队列；左摇杆死区 780→480；游戏线程 AboveNormal 优先级。
- **射击模式图标透明度**：PNG 静态图标统一走 `GamepadActionIconImage`，响应全局/单键透明度与按下高亮。

## [0.2.5] - 2026-06-13

### 新增

- **治疗 / 投掷轮盘**：长按弹出四向选择（松手确认）；短按仍快速使用默认项（注射器 / 破片）。
- **战术技能瞄准**：长按战术键进入 LB + 瞄准，可在按钮或全屏拖动调整准星；短按快速释放。

### 改进

- **奔跑**：上推移动摇杆冲刺，**松手即停**，不再锁定持续冲刺。
- **开镜（ADS）**：改为点按切换，无需长按。
- **绝招**：映射为 LB + RB 组合键（与 ping 区分）。
- **瞄准**：修正水平方向；默认灵敏度微调。

## [0.2.4] - 2026-06-15

### 改进

- **默认布局**：采用左上火力 + 右半屏战斗区布局（与当前推荐键位一致）。
- **编辑/设置入口**：移至右上角，默认键位为顶部按钮留出空间。

### 修复

- **右半屏滑动瞄准失效**：移动轮盘不再全屏拦截触摸，右半屏空白区域可正常转动视角。

---

## [0.2.3] - 2026-06-15

### 改进

- **默认键位**：开镜（ADS）与战术技能（TACTICAL）移至右上角，更符合拇指操作习惯。
- **按键尺寸**：编辑布局时大小滑条支持 **64%～250%**（以 56dp 为 100%），射击等主键可放大至 140dp。

---

## [0.2.2] - 2026-06-15

### 修复

- **布局编辑重复按钮**：旧版 Xbox 键位（RT/LT/A/B…）与语义键（开火/开镜/跳跃…）合并去重，加载与保存时自动规范化。
- **布局拖动卡顿**：修复拖动时 `pointerInput` 因坐标变化反复重建导致只能移动一小段的问题，改用稳定手势识别。
- **Broken pipe 断联（加强）**：
  - WebSocket 发送串行化，避免并发 write；
  - 断线重连后恢复手柄模式，减少 `input_mode` 反复切换；
  - 编辑布局时暂停手柄数据上报；
  - 重连防抖延长至 5 秒，避免与定时重连叠加；
  - 手柄模式下 RTT 探测间隔放宽至 5 秒。

---

## [0.2.1] - 2026-06-15

### 修复

- **连接频繁断联（Broken pipe）**：合并自动重连与 `onResume` 重连，3 秒内防抖，避免双连接互相踢线。
- **射击模式空闲流量**：手柄状态无变化时不再重复发送全零帧，降低 WiFi 与 PC 负载。

---

## [0.2.0] - 2026-06-13

### 新增

- **射击游戏模式**：横屏虚拟 Xbox 360 手柄，左半屏移动、右半屏滑动瞄准，可自定义按键布局（Apex 默认预设）。
- **PC 端 ViGEm 集成**：通过 ViGEmBus 向游戏输出虚拟手柄；新增 `gamepad` WebSocket 协议与独立发送线程。
- **Android 游戏界面** (`GamepadScreen`)：纯黑沉浸画布、横屏锁定、离线可编辑布局。
- **自定义图标**：开火、开镜、跳跃、蹲下、换弹、战术、绝招、投掷、治疗、互动、切枪、地图、背包等圆形按键图标（`icon/` 源图 → `drawable-nodpi`）。
- **灵敏度体系**：
  - 腰射 / 开镜 **水平、垂直分别调节**（开镜按住 LT 时自动切换）；
  - **滑动加速度**（快速甩枪/转身加成）；
  - 瞄准平滑、移动死区、松手衰减、移动/瞄准分项设置。
- **移动轮盘**：固定左下轮盘或浮动轮盘；上推锁定奔跑（LS）。
- **游戏内 RTT 显示**：顶部居中低调显示延迟与丢包（仅已连接且非编辑模式）。
- **文档**：[`docs/GAMEPAD-MODE.md`](docs/GAMEPAD-MODE.md) 射击模式完整使用说明。

### 改进

- 圆形图标显示与触摸命中（`CircleShape` 裁剪，无方角白边）。
- 设置页文案与逻辑：腰射/开镜分组、松手衰减实际生效、恢复布局不丢失草稿、退出游戏模式同步界面状态。
- 保存设置时 `AppSettings.clamped()` 收拢滑条合法区间。
- 瞄准引擎：分段加速曲线、高灵敏度上限（水平最高 12）、小屏转身优化。

### 修复

- 瞄准「停指衰减」滑条原先不生效（松手即清零）的问题。
- 「恢复 Apex 默认布局」会覆盖未保存设置草稿的问题。

### 依赖

- PC：`vigem-client` 等（见 `pc/Cargo.toml`）。
- 用户 PC 需自行安装 **ViGEmBus** 驱动。
